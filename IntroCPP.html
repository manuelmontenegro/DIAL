Introducción

Principales diferencias entre C++ y Python

  • C++ es un lenguaje estáticamente tipado.
    • Hay que declarar cada variable de antemano.
    • Hay que indicar el tipo de cada variable (entero, real, booleano, etc.)
  • Sintaxis distinta.
    • Uso de ; al final de una sentencia.
    • En lugar de utilizar indentación, se utilizan { y } para indicar bloques de código.

Primer programa en C++

#include <iostream>

int main() {
  int x;
  std::cout << "Introduce un número: ";
  std::cin >> x;
  std::cout << "Has introducido el número " << x << "\n";
  return 0;
}

Principales elementos

  1. Función principal:

    int main() {
      ...   // El contenido de esta función se ejecutará
            // al empezar el programa
    }

  2. Declaración de variables:

    int x;   // La variable x es de tipo entero

  3. Escritura por pantalla:

    // '\n' indica salto de línea
    std::cout << "Hola\n";
    std::cout << "Has introducido " << x << "\n";

Principales elementos

  1. Lectura de teclado:

    // Lee un número de la entrada y lo guarda en la variable x
    cin >> x;

La directiva using namespace

  • Permite ahorrarnos el uso de std:: antes de cada variable y tipo definido en la biblioteca estándar de C++.

#include <iostream>

using namespace std;

int main() {
  int x;
  cout << "Introduce un número: ";
  cin >> x;
  cout << "Has introducido el número " << x << "\n";
  return 0;
}

Tipos disponibles

Principales tipos

  • int: números enteros
  • bool: valores booleanos
  • float, double: valores en coma flotante
  • char: caracteres
  • string: cadenas de caracteres
    • Requiere incluirlas mediante #include <string>
  • vector<T>: listas de elementos de tipo T
    • Requiere incluirlas mediante #include <vector>

Ejemplos

int x = 3;

bool b = true;     // valores booleanos: true o false

float f = 2.45;

char c = '3';      // literales char: entre comillas simples (')

string s = "Hola"; // literales cadena: entre comillas dobles (")

vector<int> v = {1, 4, 10, 2, -3};

Sentencias en C++

Sentencias en C++

  • Asignación e inicialización:

    int x = 10;
    int y = 2;

    x = x + 1;
    y = 4 * y + x;

  • Incremento de variables:

    x++;    // equivale a x = x + 1;
    x--;    // equivale a x = x - 1;

Sentencias en C++

  • Condicionales:

    if (x == 0) {
      z = 3 + 3;
    }

    if (x > 10) {
      x--;
    } else {
      x++
    }

  • Bucles while:

    int n = 1;
    while (x >= 0) {
      n = n * 2;
      x--;
    }

Sentencias en C++

  • Bucles for:

    int suma = 0;  
    for (int i = 0; i < 10; i++) {  // La variable i recorre los valores desde 0 hasta 9
      suma += i;
    }

    int suma = 0;  
    for (int i = 10; i > 0; i--) {  // La variable i recorre los valores desde 10 hasta 1
      suma += i;
    }

Vectores

Inicialización de vectores

  • Indicando sus elementos en la inicialización:

    vector<char> vocales = {'a', 'e', 'i', 'o', 'u'};
    cout << vocales.size() << '\n';   // Imprime 5
    cout << vocales[2] << '\n';       // Imprime i

  • Creando un vector vacío y usando push_back:

    vector<char> vocales;
    vocales.push_back('a'); vocales.push_back('e'); // ...

  • Creando un vector con tamaño y usando asignaciones:

    vector<char> vocales(5);    // Vector de cinco elementos
    vocales[0] = 'a';
    ...
    vocales[4] = 'u';

Recorrido de un vector

  • Mediante bucles for con índice:

    for (int i = 0; i < vocales.size(); i++) {
      cout << vocales[i] << '\n';
    }

  • Mediante bucles for sin índice:

    for (char c : vocales) {
      cout << c << '\n';
    }

¡Ojo con las asignaciones a vectores!

  • Cuando se asigna un vector a otro, se crea una copia del vector origen al vector destino.

    vector<int> a = {1, 5, 3, 2};

    vector<int> b = a;

    b[0] = 3;

    cout << b[0] << '\n'; // Imprime 3
    cout << a[0] << '\n'; // Imprime 1

Funciones

Definición de una función

  • Es necesario indicar el tipo de los parámetros y el tipo del resultado:

    int suma(int x, int y) {
      return x + 2 * y;
    }

  • Si la función no devuelve ningún valor, el tipo del resultado es void:

    void saludar(int num_veces) {
      for (int i = 0; i < num_veces; i++) {
        cout << "Hola\n";
      }
    }

  • Llamadas a función:

      int m = suma(3, 4);
      saludar(m);

Parámetros por valor y por referencia

Parámetros por valor

  • Por defecto, cuando llamamos a una función con unos argumentos, los parámetros de la función contendrán una copia de esos argumentos:

    void incrementar(int x) {
      x++;
    }


    int main() {
      int v = 3;

      // La función incrementar modificará una copia de la variable v.
      // La variable v original no se verá afectada.
      incrementar(v);

      cout << v << '\n';    // Imprime: 3
    }

Parámetros por referencia

  • Si queremos que los cambios que se hagan en los parámetros se reflejen en los argumentos, hemos de pasarlos por referencia.

    • Para ello, se utiliza el símbolo & antes del nombre del parámetro:

    void incrementar(int &x) {
      x++;
    }


    int main() {
      int v = 3;

      // La función incrementar modificará la variable x original!
      incrementar(v);

      cout << v << '\n';    // Imprime: 4
    }

Paso de vectores como parámetro

  • Cuando pasamos tipos complejos como parámetro (como por ejemplo, vectores), conviene pasarlos por referencia para evitar que se cree una copia en cada llamada:

    void imprimir_elementos(vector<int> &v) {
      for (int i = 0; i < v.size(); i++) {
        cout << v[i] << ' ';
      }
      cout << '\n';
    }

Paso de vectores como parámetros constantes

  • Si la función no va a modificar el vector, conviene pasarlo como referencia constante.

    • De este modo, el compilador no nos dejará modificarlos desde dentro de la función.

    void imprimir_elementos(const vector<int> &v) {
      for (int i = 0; i < v.size(); i++) {
        cout << v[i] << ' ';
      }
      cout << '\n';
    }

Registros (structs)

Definición de un struct

  • Un struct en C++ es un tipo de dato definido por el usuario que agrupa varias variables (llamadas atributos) bajo un mismo nombre.

    struct Punto {
      int x;
      int y;
    };    // <-- ojo con el punto y coma al final de la declaración;

    struct Persona {
      string nombre;
      string dni;
      int edad;
    };

Uso de structs

Punto p1;
p1.x = 10;
p1.y = 20;

// Inicialización alternativa
Punto p1 = {10, 20};

Asignación de un struct y paso por parámetro

  • Al igual que con los vectores, asignar una variable de tipo struct a otra provoca una copia de la variable origen:

    Punto p1;
    p1.x = 10; p1.y = 20;

    Punto p2 = p1;
    p2.x = 20;

    cout << p1.x << '\n';  // Imprime '10'

Asignación de un struct y paso por parámetro

  • Lo mismo ocurre cuando pasamos struct como parámetros. Por defecto, se pasan por valor:

    void desplazar_derecha(Punto p) {  p.x++; }

    int main() {
      Punto p1 = {1, 4};
      desplazar_derecha(p1);
      cout << p1.x << '\n';   // Imprime '1'
    }

Paso por referencia

  • Si queremos que la función modifique realmente el argumento, debemos pasar el parámetro por referencia:

    void desplazar_derecha(Punto &p) {
      p.x++;
    }

    int main() {
      Punto p1 = {1, 4};
      desplazar_derecha(p1);
      cout << p1.x << '\n';   // Imprime '2'
    }
